テスト以外で初めてのgit hub公開作品です

GCPでspreadsheet apiなどの設定、KEY.jsonファイルをダウンロードする必要があります。
スプレッドシートはブクログでダウンロードして少し整形する必要があります。
caril APIも取得する必要があります。

script.jsは更新したいgoogleスプレッドシートのApps scriptのエディタにコードをペーストして利用します。

herokuファイルはherokuにプッシュして利用します。
